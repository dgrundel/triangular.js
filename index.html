Triangular.js

Two-way databinding is sooo 2014.

Download .zip Download .tar.gz View on GitHub

What's that you say? Three-way data binding?

Form data seamlessly synced between your JavaScript, the DOM, and LocalStorage.

Oh yeah, that means it syncs across browser tabs. Yes, really.

Even works on Grandpa's browser.

triangular.js should work for IE7+. Of course, the real browsers are supported, too.

This paragraph is bound to the input box below. Try changing it!

These radio buttons are also bound to the same input. Try typing 'car', 'bike', or 'truck' into the input box. (All lowercase to match the values!)



And wouldn't you know, this select box is all bound up, too. Try it out.

Checkboxes work, as well, but a little differently as you might expect. If your value is truthy, they'll be checked. If your value is falsey or the string "false", it will be unchecked.

Try clearing out the text field to uncheck the box!
